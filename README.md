# PhilTreeCrawler

PhilTreeCrawler is an interactive, procedurally generated graph crawler for philosophical exploration. It allows users to traverse a web of philosophical concepts, where each node provides a lesson and branches into related ideas.

## Features

- **Procedural Lesson Generation**: Fetches or generates unique lessons for each concept encountered.
- **Dynamic Graph Expansion**: New branches are added to the global graph as users interact with them.
- **Persistent Knowledge Base**: Stores generated content and graph relationships for future traversal.
- **Guided Exploration**: Offers multiple paths from each concept, detailing their relationship to the current topic.
